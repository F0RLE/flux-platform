# Contributing

**Flux Platform is Proprietary Software.**

We appreciate your interest, but we **do not accept** pull requests, code contributions, or modifications from the public at this time.

This repository is for distribution and issue tracking only.
The source code is provided for reference (if visible) but is protected by copyright.
Unauthorized modification or redistribution is prohibited.

## Reporting Issues

If you encounter bugs or have feature requests, please open an Issue.
We welcome detailed bug reports!

1.  Check existing issues to avoid duplicates.
2.  Provide your OS version and Flux Platform version.
3.  Attach logs if possible (make sure to redact sensitive info like tokens).

Thank you for understanding.
