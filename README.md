## 🎯 这是什么？

**Flux Platform** 是一个通用的服务管理平台，旨在简化模块化应用程序的部署、监控和管理。它提供了一个现代化的 Web 界面，用于通过统一的仪表板处理服务、监控系统资源和查看日志。

## ✨ 主要功能

- 🚀 **Web启动器** - 现代化的 Web 界面，用于管理所有组件
- 📊 **实时监控** - CPU、GPU、RAM、VRAM、磁盘、网络，带颜色指示
- 🔄 **模块管理** - 可扩展的模块系统，支持安装和管理自定义服务
- 🛡️ **安全** - 速率限制、数据验证
- 📋 **日志** - 详细的系统和交互日志
- 🛠️ **自动化** - 自动处理环境配置和依赖项

## 📋 要求

- Windows 10/11
- Python 3.10 - 3.14（自动安装）
- 互联网连接（用于下载依赖项）

## 🛠️ 快速开始

### 安装

1. **运行 `Launch.vbs`**
   - 脚本将自动安装 Python、Git 和所有必需的依赖项
   - 安装可能需要几分钟
   - 将打开带有图形界面的启动器

## 📁 项目结构

```
flux-platform/
├── config/                # 应用程序配置
├── Launch.vbs             # 主启动脚本
├── src/                   # 主源代码
│   ├── backend/           # 后端 (Python)
│   └── frontend/          # 前端 Web 界面
├── LICENSE                # 许可证
└── README.md              # 文档
```

## 🎯 Что это?

**Flux Platform** — это универсальная платформа управления сервисами, предназначенная для упрощения развертывания, мониторинга и управления модульными приложениями. Она предоставляет современный веб-интерфейс для работы с сервисами, мониторинга системных ресурсов и просмотра логов через единую панель управления.

## ✨ Основные возможности

- 🚀 **Веб-лаунчер** - Современный веб-интерфейс для управления всеми компонентами
- 📊 **Мониторинг в реальном времени** - CPU, GPU, RAM, VRAM, Disk, Network с цветовой индикацией
- 🔄 **Управление модулями** - Расширяемая система модулей для установки и управления сервисами
- 🛡️ **Безопасность** - Rate limiting, валидация данных
- 📋 **Логирование** - Детальные логи системы и взаимодействий
- 🛠️ **Автоматизация** - Автоматическая настройка окружения и зависимостей

## 📋 Требования

- Windows 10/11
- Python 3.10 - 3.14 (устанавливается автоматически)
- Интернет-соединение (для загрузки зависимостей)

## 🛠️ Быстрый старт

### Установка

1. **Запустите `Launch.vbs`**
   - Скрипт автоматически установит Python, Git и все необходимые зависимости
   - Установка может занять несколько минут
   - Откроется лаунчер с графическим интерфейсом

## 🎯 What is this?

**Flux Platform** is a universal service management platform designed to simplify the deployment, monitoring, and management of modular applications. It provides a modern web interface for handling services, monitoring system resources, and viewing logs through a unified dashboard.

## ✨ Key Features

- 🚀 **Web Launcher** - Modern web interface for managing all components
- 📊 **Real-time Monitoring** - CPU, GPU, RAM, VRAM, Disk, Network with color indicators
- 🔄 **Module Management** - Extensible module system for installing and managing services
- 🛡️ **Security** - Rate limiting, data validation
- 📋 **Logging** - Detailed logs for system and interactions
- 🛠️ **Automation** - Automated environment setup and dependency management

## 📋 Requirements

- Windows 10/11
- Python 3.10 - 3.14 (installed automatically)
- Internet connection (for downloading dependencies)

## 🛠️ Quick Start

### Installation

1. **Run `Launch.vbs`**
   - The script will automatically install Python, Git, and all necessary dependencies
   - Installation may take several minutes
   - The launcher with graphical interface will open
